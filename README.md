# Jeu de Catégories

## Description
Un jeu de catégories interactif développé en JavaScript.

## Fonctionnalités
- Interface utilisateur moderne
- Système de gestion des mots
- Timer intégré
- Gestion des sons
- Système de conseils

## Installation
1. Cloner le repository
2. Ouvrir `index.html` dans un navigateur web
